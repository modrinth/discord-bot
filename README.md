# Modrinth Discord Bot

 The discord bot that powers our Discord server.

## Environment variables

See .env.example

## Development

1. Clone the repository
2. Copy .env.example and rename it into .env
3. Install deps and run in dev mode:

```powershell
$ pnpm install
pnpm dev
```

Format and lint:

```powershell
pnpm run fix
```
